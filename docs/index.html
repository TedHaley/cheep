What is it?

cheep is a tiny, single-binary CLI for multi-agent coding. One expensive, capable orchestrator (Claude) breaks your task into pieces, hands each to a free, local executor (Qwen via Ollama), then verifies the work and recovers any executor that gets stuck — loops, runs out of context, or errors out.

Orchestrator — Claude
Plans, delegates, and verifies. Reads files and runs tests to check the work. Never edits code itself.
Executors — local Qwen
Do the actual editing and running. Free and unlimited on your own hardware.

Quick start

# 1. Run a local model with an OpenAI-compatible endpoint
ollama serve &
ollama pull qwen2.5-coder

# 2. Give Claude an API key
export ANTHROPIC_API_KEY=sk-ant-...

# 3. Check both ends are reachable
cheep check

# 4. Hand it a task
cheep run "add a /health endpoint and a test for it" --workdir .

Configuration

Everything is environment variables, with sane defaults (Claude Sonnet + local Qwen on Ollama):
